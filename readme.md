# Smart Finance Telegram Bot

Telegram-бот для учёта доходов и расходов, анализа финансов и контроля бюджета.

## Функции
- Добавление доходов и расходов
- Подсчёт баланса
- Анализ категорий
- Финансовая статистика
- И многие другие

## Технологии
- Python
- SQLite
- Telegram Bot API
- Railway
- И многие другие
